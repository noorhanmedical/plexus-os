- Left panel: Patient search input + clickable list of matching patients.
- Middle panel: Prescreens list for the selected patient.
- Right panel: Prescreen detail editor with Save button.

- Base URL: I will paste it into api.js as API_URL.
- API key: I will paste it into api.js as API_KEY.
- All requests must include header: x-api-key: API_KEY.
- GET endpoints use query params: ?action=...
- POST endpoints send JSON: { action: "...", ... }.

Endpoints to use:
GET:
- action=patients.search with params { query, limit }
- action=patients.get with params { patient_uuid }
- action=prescreen.list with params { status (optional), limit }

POST:
- action=prescreen.get with body { prescreen_id } (or allow patient_uuid + requested_ancillary_code)
- action=prescreen.update with body { prescreen_id, updates: { scheduled_datetime, assigned_staff, location, notes } }
- action=prescreen.updateStatus with body { prescreen_id, status }  (this sets eligibility_status_final in Sheets)
- action=prescreen.create with body { patient_uuid, requested_ancillary_code }

3) app.js:

       scheduled_datetime, assigned_staff, location, notes
